16:T3b50,<p>Bismillah, hari ini kita akan membahas mengenai Combinator di CSS. Apa itu Combinator? Combinator merupakan suatu karakter yang menyatakan hubungan antara selector, untuk menargetkan element dalam dokumen. Misalnya untuk menyeleksi semua element <code>&#x3C;p></code> yang ada di dalam element <code>&#x3C;article></code>, kita menggunakan Descendant Combinator ( ):</p>
<pre><code class="language-css">article p { }
</code></pre>
<p>Selain Descendant Combinator ( ), ada juga Child combinator (>), Adjacent sibling combinator (+) dan General sibling combinator (~).</p>
<p>Sebelum lanjut, seperti biasa kamu harus sudah memahami dasar-dasar CSS, terutama CSS Selector. Jika belum, kamu bisa mempelajarinya di <a href="https://youtube.com/playlist?list=PLFIM0718LjIUBrbm6Gdh6k7ZUvPIAZm7p">playlist CSS Dasar Web Programming Unpas</a>.</p>
<p>Berikut penjelasan dari masing-masing combinator:</p>
<h2>Descendant Combinator (Kombinator Keturunan)</h2>
<p>Descendant Combinator adalah combinator yang menggunakan satu spasi ( ) sebagai penghubung diantara selector nya, contoh:</p>
<pre><code class="language-css">.reza p {  
  color: red;  
}
</code></pre>
<p>Arti dari script css diatas adalah, <em>seleksi semua element <code>&#x3C;p></code> yang merupakan descendant (keturunan) dari element dengan class "reza", lalu ubah warna textnya menjadi merah</em>.</p>
<blockquote>
<p>Suatu element bisa dikatakan descendant (keturunan) dari element lain adalah ketika element tersebut berada di dalam element lain. Misalnya kita punya element <code>&#x3C;div></code> dan element <code>&#x3C;p></code>, supaya element <code>&#x3C;p></code> bisa dikatakan descendant (keturunan) dari element <code>&#x3C;div></code>, maka tempatkan element <code>&#x3C;p></code> di dalam element <code>&#x3C;div></code>. Tidak masalah jika element <code>&#x3C;p></code> berada di dalam element lain, element <code>&#x3C;article></code> misalnya, intinya adalah selama element <code>&#x3C;p></code> tersebut barada di dalam element <code>&#x3C;div></code>, maka element <code>&#x3C;p></code> bisa dikatakan descendant (keturunan) dari element <code>&#x3C;div></code>. Keadaan dimana suatu element berada di dalam element lain biasa disebut dengan <em>nested element</em> atau element bersarang.</p>
</blockquote>
<p>Berikut contoh implementasinya:</p>
<pre><code class="language-html">&#x3C;div class="reza">  
  &#x3C;p>Hay saya programmer&#x3C;/p>
  
  &#x3C;div class="dian">  
    &#x3C;p>Hay salam kenal&#x3C;/p>  
    &#x3C;p>Selamat datang&#x3C;/p>
    &#x3C;div class="hallo">  
        &#x3C;p>Hallo apa kabar&#x3C;/p>  
    &#x3C;/div>  
  &#x3C;/div> 
   
&#x3C;/div>
&#x3C;p>Saya biasa&#x3C;/p>
</code></pre>
<pre><code class="language-css">.reza p {  
  color: red;  
}
</code></pre>

Jika kamu lihat element <code>&#x3C;p></code> yang berada diluar element <code>&#x3C;div></code> dengan class "reza" warna text-nya tidak berubah menjadi merah, karena element <code>&#x3C;p></code> tersebut bukan descendant (keturunan) dari element <code>&#x3C;div></code> dengan class "reza".</p>
<h2>Child Combinator (Kombinator Anak)</h2>
<p>Child Combinator adalah combinator yang menggunakan karakter lebih besar dari (>) sebagai penghubung diantara selectornya, contoh:</p>
<pre><code class="language-css">.reza > p {  
  color: red;  
}
</code></pre>
<p>Arti dari script diatas adalah, <em>seleksi semua element <code>&#x3C;p></code> yang termasuk child (anak) dari element dengan class "reza" lalu ubah warna textnya menjadi merah</em>.</p>
<blockquote>
<p>Suatu element bisa dikatakan child (anak) dari element lain adalah ketika element tersebut berada di dalam elemen lain dan element tersebut tidak berada di dalam element lain lagi. Misalnya kita punya element <code>&#x3C;div></code> dan element <code>&#x3C;p></code>, supaya element <code>&#x3C;p></code> bisa dikatakan child (anak) dari element <code>&#x3C;div></code>, maka element <code>&#x3C;p></code> harus berada di dalam element <code>&#x3C;div></code> dan tidak boleh berada di dalam element lain lagi, element <code>&#x3C;article></code> misalnya. Dengan kata lain, walaupun kita menempatkan element <code>&#x3C;article></code> yang di dalamnya ada element <code>&#x3C;p></code> di dalam element <code>&#x3C;div></code>, element <code>&#x3C;p></code> bukanlah child (anak) dari element <code>&#x3C;div></code>, melainkan cucu dari element <code>&#x3C;div></code>, sedangkan element <code>&#x3C;article></code>-lah yang merupakan child (anak) dari element <code>&#x3C;div></code>.</p>
</blockquote>
<p>Berikut contoh implementasinya:</p>
<pre><code class="language-html">&#x3C;div class="reza">  
  &#x3C;p>Hay saya programmer&#x3C;/p>
  
  &#x3C;div class="dian">  
    &#x3C;p>Hay salam kenal&#x3C;/p>  
    &#x3C;p>Selamat datang&#x3C;/p>  
  &#x3C;/div>
  
  &#x3C;p>Saya biasa&#x3C;/p>  
&#x3C;/div>  
&#x3C;p>Semoga bermanfaat&#x3C;/p>
</code></pre>
<pre><code class="language-css">.reza > p {  
  color: red;  
}
</code></pre>

<h2>Adjacent Sibling Combinator (Kombinator Saudara Kandung Terdekat)</h2>
<p>Adjacent Sibling Combinator adalah combinator yang menggunakan karakter plus (+) sebagai penguhubung diantara selectornya, contoh:</p>
<pre><code class="language-css">.reza + p {  
  color: red;  
}
</code></pre>
<p>Arti dari script diatas adalah, <em>seleksi element <code>&#x3C;p></code> yang merupakan adjacent sibling (saudara kandung terdekat) dari element dengan class "reza", lalu ubah warna textnya menjadi merah</em>.</p>
<blockquote>
<p>Suatu element dikatakan adjacent sibling (saudara kandung terdekat) dari element lain adalah ketika element tersebut berada di posisi setelah elemen lain dan element yang menjadi adjacent sibling (saudara kandung terdekat) hanya element yang pertama. Misalnya kita punya beberapa element, yaitu: element <code>&#x3C;p></code> dengan id 1, element <code>&#x3C;article></code>, element <code>&#x3C;p></code> dengan id "2" dan element <code>&#x3C;p></code> dengan id 3. Supaya element <code>&#x3C;p></code> dengan id 1 bisa dikatakan adjacent sibling (saudara kandung terdekat) dari element <code>&#x3C;article></code>, maka kita harus menempatkan element <code>&#x3C;p></code> dengan id 1  pada urutan dari atas ke bawah, tepat setelah element <code>&#x3C;article></code>, sedangkan element <code>&#x3C;p></code> yang lain bebas ditempatkan di mana saja, di atas element <code>&#x3C;article></code> atau di bawah element <code>&#x3C;p></code> dengan id 1. Jadi misalnya kita buat urutan element-element tersebut dari atas ke bawah menjadi, pertama element <code>&#x3C;p></code> dengan id 2, element <code>&#x3C;article></code>, element <code>&#x3C;p></code> dengan id 1 dan element <code>&#x3C;p></code> dengan id 3. Dengan urutan tersebut element <code>&#x3C;p></code> dengan id 1 merupakan adjacent sibling (saudara kandung terdekat) dari element <code>&#x3C;article></code>, sedangkan element <code>&#x3C;p></code> dengan id 2 dan 3 bukanlah adjacent sibling (saudara kandung terdekat) dari element <code>&#x3C;article></code>.</p>
</blockquote>
<p>Berikut contoh implementasinya:</p>
<pre><code class="language-html">&#x3C;div class="dian">  
  &#x3C;p>Hello Dunia&#x3C;/p>
  
  &#x3C;div class="reza">  
    &#x3C;p>Hay saya programmer&#x3C;/p>  
    &#x3C;p>Saya biasa&#x3C;/p>  
  &#x3C;/div>
  
  &#x3C;p>Semoga bermanfaat&#x3C;/p>  
  &#x3C;p>Hay salam kenal&#x3C;/p>  
&#x3C;/div>
</code></pre>
<pre><code class="language-css">.reza + p {  
  color: red;  
}
</code></pre>

Lalu, element <code>&#x3C;p></code> dengan text <em>Hay salam kenal</em> juga tidak berubah menjadi merah, memang posisi element <code>&#x3C;p></code> dengan text <em>Hay salam kenal</em> berada setelah element <code>&#x3C;div></code> dengan class "reza", tetapi urutannya tidak berada pada urutan pertama.</p>
<h2>General Sibling Combinator (Kombinator Saudara Kandung Umum)</h2>
<p>General Sibling Combinator adalah combinator yang menggunakan karakter tilde ( ~ ) sebagai penghubung diantara selectornya, contoh:</p>
<pre><code class="language-css">.reza ~ p {
  color: red;  
}
</code></pre>
<p>Arti dari script diatas adalah, <em>seleksi semua element <code>&#x3C;p></code> yang merupakan general sibling (saudara kandung umum) dari element dengan class "reza", lalu ubah warna textnya menjadi merah</em>.</p>
<blockquote>
<p>Cara kerja dari general sibling combinator (kombinator saudara kandung umum) mirip dengan cara kerja adjacent sibling combinator (kombinator saudara kandung terdekat), hanya saja yang membedakan adalah semua element yang berada di posisi setelah element lain adalah general sibling (saudara kandung umum) dari element lain tersebut. Misalnya (dengan contoh yang sama), kita punya beberapa element, dengan urutan: element <code>&#x3C;p></code> dengan id 1, element <code>&#x3C;article></code>, element <code>&#x3C;p></code> dengan id 2 dan element <code>&#x3C;p></code> dengan id 3. Dengan urutan tersebut yang termasuk general sibling (saudara kandung umum) dari element <code>&#x3C;article></code> adalah element <code>&#x3C;p></code> dengan id 2 dan element <code>&#x3C;p></code> dengan id 3.</p>
</blockquote>
<p>Berikut contoh implementasinya:</p>
<pre><code class="language-html">&#x3C;div class="dian">  
  &#x3C;p>Hello Dunia&#x3C;/p>
  
  &#x3C;div class="reza">  
    &#x3C;p>Hay saya programmer&#x3C;/p>  
    &#x3C;p>Saya biasa&#x3C;/p>  
  &#x3C;/div>
  
  &#x3C;p>Semoga bermanfaat&#x3C;/p>  
  &#x3C;p>Hay salam kenal&#x3C;/p>  
&#x3C;/div>
</code></pre>
<pre><code class="language-css">.reza ~ p {  
  color: red;  
}
</code></pre>
